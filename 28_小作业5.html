CSS(层叠样式表)

层叠样式表 (Cascading Style Sheets，缩写为 CSS），是一种 样式表 语言，用来描述 HTML 或 XML（包括如 SVG、MathML、XHTML 之类的 XML 分支语言）文档的呈现。CSS 描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被渲染的问题。

使用编程语言，制定了一系列指令，即计算机完成任务所需执行的步骤。有更接近机器码而远离人类语言的语言（如 Assembly），也有更接近人类语言而远离机器码的高级编程语言（如 C# 和 Java）。编程语言数不胜数，而且它们的数量只会增加，所以总是有新的东西可以谈论，新的库可以探索。可以通过多种方式对它们进行分类——例如，有面向对象的编程语言、函数式语言，或者我们可以根据它们的用途进行分类。

  • SS 预处理器是可以生成 CSS 代码的 CSS 扩展，这意味着可以使用编程方法自动执行步骤。预处理器有很多，但最著名的两个是 Sass 和 Less。
  • 这些有助于创建和维护复杂应用程序和网站的 CSS 结构，并创建可由多人编辑的透明代码。有了它们，就可以在 CSS 中使用嵌入、循环和继承。
  • 在CSS中，可以声明函数，使用面向对象，构建一个透明的、易于维护的模块化系统。
